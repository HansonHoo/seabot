Description détaillée

Ensemble des éléments spécifiques à une plateforme utilisés par TIS.

Les éléments spécifiques à une plateforme sont principalement les types variables, la librairie utilise les types POSIX, il faut donc les redéfinir sur les plateforme qui ne sont pas POSIX.

Les implémentations des fonctions sont présente dans /plateformes/src.
