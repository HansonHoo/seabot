uint8_t *  pin 
)

Envoie la commande AT+CPIN au modem.

La commande AT+CPIN permet d'envoyer le code PIN de la carte SIM. Cette fonction ne retourne pas une erreur si le code PIN est incorrect.

Paramètres:
[in]propertiesPointeur vers la structure de type TIS_properties contenant les propriétés de la librairie.
[in]pinLe pointeur vers la chaine de caractère contenant le code PIN.
Renvoie:
Un code d'erreur.

int32_t TIS_AT_SBDD ( TIS_properties *  properties,
bool  MO_buffer,
bool  MT_buffer 
)

Envoie la commande AT+SBDD au modem.

La commande AT+SBDD (Short Burst Data: Clear SBD Message Buffer(s)) permet de vider les tampons SBD du modem.

Paramètres:
[in]propertiesPointeur vers la structure de type TIS_properties contenant les propriétés de la librairie.
[in]MO_bufferSi TRUE, vide le tampon des messages sortants. Si false, ne vide pas le tampon de messages sortant
[in]MT_bufferSi TRUE, vide le tampon des messages entrants. Si false, ne vide pas le tampon de messages entrant.
Renvoie:
Un code d'erreur.
Voir également:
TIS_error.h

int32_t *  MT_status,

)

Envoie la commande AT+SBDI au modem.

La commande AT+SBDD (Short Burst Data: Initiate an SBD Session) permet de lancer une session SBD.

Paramètres:
[in]propertiesPointeur vers la structure de type TIS_properties contenant les propriétés de la librairie.
[out]MO_statusPointeur vers la variable où sera écrite le statut de l'envoie du message sortant.
[out]MT_statusPointeur vers la variable où sera écrite le statut de réception du message entrant.
[out]MT_queuedPointeur vers la variable où sera écrite le nombre de message entrant en attente dans le satellite.

Un code d'erreur.

int32_t TIS_AT_SBDRB ( TIS_properties *  properties,
uint8_t *  message,
int32_t *  MT_length 
)

Envoie la commande AT+SBDRB au modem.

La commande AT+SBDRB (Short Burst Data: Read Binary Data from ISU) permet de récupérer le message entrant contenue dans le tampon du modem.

Paramètres:
[in]propertiesPointeur vers la structure de type TIS_properties contenant les propriétés de la librairie.
[out]messageUn pointeur vers le tampon où sera écrit le message. Ce tampon doit être de taille TIS_SBD_SIZE_MAX octets.
[out]MT_lengthPointeur vers la variable où sera écrite la taille du message entrant.
Renvoie:
le nombre d'octet écrits dans message.

int32_t TIS_AT_SBDWB ( TIS_properties *  properties,
uint8_t *  message,
int32_t  count 
)

Envoie la commande AT+SBDWB au modem.

La commande AT+SBDWB (Short Burst Data: Write Binary Data to the ISU) permet d'envoyer un message sortant vers le modem.

Paramètres:
[in]propertiesPointeur vers la structure de type TIS_properties contenant les propriétés de la librairie.
[in]messageUn pointeur vers le tampon où est stocké le message. ce tampon doit être de taille TIS_MODEM_SBD_SIZE_MAX octets ou inférieur.
[in]countLe nombre d'octets à envoyer.
Renvoie:
le nombre d'octets envoyés avec succès.
